# Tech News Pulse

Tech News Pulse is a web app that aggregates the latest technology and startup news in one place. It is a beginner-friendly project to practice working with APIs, search, filtering, saved items, and theme persistence using `localStorage`.

## Features

- Browse latest tech and startup news
- Filter news by category
  - AI
  - Crypto
  - Gadgets
- Search for articles by keyword
- "Read More" links for full article access
- Save articles for later reading
- Image fallback for articles with no images
- Dark Mode toggle
- Persistent theme using `localStorage`
- Responsive UI

## Tech Stack

- HTML
- CSS
- JavaScript
- NewsAPI

## Project Goal

This project was built to practice practical frontend skills:

- API integration with `fetch`
- Using the DOM to show data
- Search and category buttons
- Saving arrays in `localStorage`
- Switching themes

## How to use

1. Open `index.html` in your browser
2. Enter your free NewsAPI key in the yellow box
3. Browse, search, and save articles!

## API Used

**NewsAPI**  
`https://newsapi.org/`

> Note: You need your own free API key from NewsAPI to run this project properly. If you don't enter one, it will show a few demo articles instead.
